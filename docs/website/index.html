使用说明

  1. 下载扩展包（CRX或ZIP文件）
  2. 在Chrome浏览器中打开扩展管理页面 (chrome://extensions/)
  3. 启用开发者模式
  4. 将下载的扩展文件拖放到浏览器窗口中安装
  5. 点击工具栏中的扩展图标，选择您喜欢的颜色主题
